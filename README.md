# Phishing
Phishing attacks are one of the most common cyber threats used to steal sensitive user information. These attacks often use fake websites that appear legitimate. This project proposes an AI-based phishing detection system that analyses website features and URLs to identify and block phishing websites, helping users stay safe online.
